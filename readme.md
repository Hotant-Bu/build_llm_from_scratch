## 从零构建大语言模型
    
        从零构建大语言模型，学习原理与实践，使用PyTorch构建神经网络，训练预测从词元化到词汇表构建，
        再到手动实现神经网络，训练，计算损失，微调，实现可以预测有意义的下一句。
